Before running the applications:
Steps to download mysql-connector-java:
    1. Go to https://dev.mysql.com/downloads/connector/j/8.0.html
    2. Choose Platform Independent
    3. Download the zip file
    4. Unzip -> look for 'mysql-connector-java-8.0.30.jar'
    5. Move to JRE/lib/ext
    6. If you have any problem: https://www.javatpoint.com/example-to-connect-to-the-mysql-database
If you use VSCODE:
- You should move put the 'mysql-connector-java-8.0.30.jar' file to the referenced Library for starting connect to the database.
- mysql-connector-java-8.0.30.jar is a driver that helps connect the system to the database.
If you use IntelliJ:
- You can run right away from Main
